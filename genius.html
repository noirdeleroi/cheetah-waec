Solve the equation: \[ 2^x + 8^x = 130 \] Step 1: Express all terms with the same base.
Since \(8 = 2^3\), we can write: \[ 8^x = (2^3)^x = 2^{3x} \] So the equation becomes: \[ 2^x + 2^{3x} = 130 \]
Step 2: Substitute \(y = 2^x\). Then \(2^{3x} = (2^x)^3 = y^3\), so: \[ y + y^3 = 130 \]
Step 3: Rearrange the equation. \[ y^3 + y - 130 = 0 \]
Step 4: Solve the cubic equation.
Try rational root theorem. Try \(y = 5\): \[ 5^3 + 5 = 125 + 5 = 130 \]
So \(y = 5\) is a root. Then \(2^x = 5\)
Step 5: Solve for \(x\). \[ 2^x = 5 \Rightarrow x = \log_2 5 \] Final Answer: \[ \boxed{x = \log_2 5} \]
